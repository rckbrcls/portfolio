## My first blog post

Well, I don't know if starting this blog is a good idea. First, I'm not much of a writer. Second, I'm kind of shy. And third, maybe I'm afraid I'll have nothing to say.
But well, let's try, right? Maybe I can express something worthwhile, or maybe I'll even grow fond of writing.

This is actually the first test post for the MDX features, and I think I liked the setup I put together to help me write and publish posts.
I ended up using `@next/mdx` to make MDX work nicely inside Next.js, `gray-matter` to read the frontmatter, `remark-gfm` for the nicer Markdown bits, and `remark-math` with `rehype-katex` so I can throw formulas in here whenever I need to. So the plan is to keep writing mostly in Markdown and only lean on MDX when it actually adds something useful.

Anyway, after all that rambling, thanks for making it this far, and if you want to see my next posts, check the site from time to time (I still haven't figured out a good way to notify new posts yet, maybe I'll mention them on X (rip Twitter)).

Sorry for the informality (well, it's my site lol), and see you next time! &lt;3
